# PassKeeper - Password Manager
PassKeeper is a simple password management tool built in Java, designed to store and retrieve passwords securely. This project uses PostgreSQL for storing passwords.

## Requirements

-Java 11 or higher
- PostgreSQL Database
- PostgreSQL JDBC Driver
